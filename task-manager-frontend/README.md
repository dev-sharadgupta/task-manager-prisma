## Step 1: Create React project

run: npm create vite@latest task-manager-ui
    --> Select a framework -> React
    --> Select a variant -> TypeScript

## Short Command to intall React + Typescipt
npm create vite@latest task-manager-frontend -- --template react-ts

Created a React 19 project using Vite with TypeScript template
to build the frontend for the Task Management application.

## Step 2: Navigate to frontend project
run: cd task-manager-frontend

Moved into the React frontend project directory
to continue setup and configuration.

## Step 3: Install frontend dependencies
run: npm install

Installed all npm dependencies for the React + TypeScript
project generated by Vite.


## Step 4: Install Tailwind for styling
run: npm install tailwindcss @tailwindcss/vite

-- vite.config.ts
import tailwindcss from '@tailwindcss/vite'  // Import Tailwind
@custom-variant dark (&:where(.dark, .dark *)); /* Use Theme */

export default defineConfig({
  plugins: [react(),
    tailwindcss(), // add tailwind here
  ],
})

-- src\index.css
@import "tailwindcss";  /* Import Taliwind */

## Step 5 : Install React Router
run: npm install react-router-dom@6 
use version 6 for better stability

## Step 6: Install lucide-react icon library
run: npm install lucide-react

## Step 7: Initialize shadcn/ui

Note: Before this
-- src\index.css
@import "tailwindcss";  /* Inport Taliwind */


-- Add Import Alias to TypeScript (tsconfig.json)

"compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": [
        "src/*"
      ]
    }
  }

-- Add Alias in tsconfig.app.json
"compilerOptions": {
  "paths": {
      "@/*": [
        "./src/*"
      ],
      "src/*": [
        "./src/*"
      ]
    }
}

-- vite.config.ts

 resolve: {
    alias: {
      "@": path.resolve(__dirname, "./src"),
    },
  },

export default defineConfig({
  plugins: [
    react(),
    tailwindcss(), // Add Tailwind here
  ],
  resolve: {
    alias: {
      "@": path.resolve(__dirname, "./src"),
    },
  },
})

run: npx shadcn@latest init

Initialized shadcn/ui using the CLI with React, TypeScript,
Tailwind CSS, and CSS variables. Configured component
aliases for scalable UI development.

Note: Later install shadcn component as per needed
